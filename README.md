# Freelance_Connect

Freelance Connect is a full-stack MERN micro-task marketplace built for students and local communities to offer and request small services. It provides a simple local freelancing platform where users can act as both clients and freelancers.

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes

### Freelancer Features
- Create and manage gigs
- Add title, description, price, and images
- View incoming service requests
- Accept / Reject requests
- Mark tasks as completed

### Client Features
- Browse available gigs
- Search services
- View gig details
- Send service requests
- Track request status

### Dashboard
- Ongoing tasks
- Completed tasks
- Profile management

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Redux Toolkit
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- JWT
- Bcrypt
- Multer

### Database
- MongoDB
- Mongoose
